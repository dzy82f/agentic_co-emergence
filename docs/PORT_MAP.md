# Port Map

`tychevia_runtime_v8` has been ported into a Python package structure.

- core runtime modules -> `agentic_co_emergence/core/`
- event stores and query helpers -> `agentic_co_emergence/storage/`
- reasoning, knowledge, memory, semantic engines -> `agentic_co_emergence/engines/`
- projections -> `agentic_co_emergence/projections/`
- models -> `agentic_co_emergence/models/models.py`
- personas -> `agentic_co_emergence/personas/`
- protocols -> `agentic_co_emergence/protocols/`
- runner -> `agentic_co_emergence/cli/run.py`
- utility scripts -> `examples/`
